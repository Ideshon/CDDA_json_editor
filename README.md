# CDDA JSON Mod Editor

Графический редактор модов для Cataclysm: Dark Days Ahead (0.G+).

- Редактирование объектов по схемам (mutations, items, monsters и т.д.)
- Схемы можно менять в файлах schemas (пока только до сборки)
- Поддержка вариантов, флагов, ссылок (ref_list)
- Удаление/добавление полей
- Ручное редактирование сырых JSON-структур
- Поддержка открытия одной папки мода или отдельного JSON-файла
- Темная тема
